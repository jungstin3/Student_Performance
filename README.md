Proyek ini merupakan aplikasi web berbasis machine learning yang dibuat menggunakan Streamlit.  
Aplikasi ini digunakan untuk memprediksi nilai performa siswa berdasarkan beberapa faktor akademik dan personal.

Cara Menjalankan Aplikasi
1. Install seluruh dependensi yang dibutuhkan
pip install -r app/requirements.txt
2. Jalankan file app.py pada folder app
3. Ketik "streamlit run app/app.py" pada terminal
